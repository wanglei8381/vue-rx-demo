# APP

Usage: wag [options] [value ...]

Commands:

    init [projectName]  初始化项目
    dev                 开发
    build               发布上线
    start               开发自动启动服务
    test                测试
    help [cmd]          display help for [cmd]

Options:

    -h, --help     output usage information
    -V, --version  output the version number
    -c, --cover    如果工程名存在,就覆盖掉
    -r, --remove   build之前将dist目录删掉
